# TOPIK Mate - 운영자 페이지(demo)

운영자의 로컬에 있는 문제 데이터셋(csv)을 데이터베이스에 적재하기 위한 운영자 페이지입니다.
단일 CSV 파일을 업로드하여 Firebase에 적재할 수 있습니다.

1. Git clone
```
git clone https://github.com/jwo29/test.git
```

2. 필요한 패키지 다운로드
```
npm install
```

3. 로컬 서버 구동
```
node app.js
```

4. http://localhost:3000 접속
